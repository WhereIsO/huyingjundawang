# 组员使用手册（数据备份工具）

本手册面向小组其他同学，只讲怎么拿到、怎么跑、怎么演示，不涉及编译和代码。照着做即可。

---

## 一、你会拿到什么

一个压缩包：`数据备份工具_可分发.zip`（约 41 MB）。

里面已经打包好**程序本体 + 全部运行库**，你的电脑**不需要**装 Qt、不需要装 Anaconda、不需要装 Visual Studio，解压就能用。

解压后文件夹长这样：

```text
数据备份工具/
  BackupTool.exe            ← 程序本体
  启动-数据备份工具.bat       ← 双击这个来启动（推荐）
  看我-使用说明.txt          ← 一页纸速查
  Qt5*.dll、platforms\ 等    ← 运行库，别删、别单独挪动
```

> 重要：不要把 `BackupTool.exe` 单独拷出来用。它要和旁边那堆 dll、`platforms\` 文件夹在一起才能跑。要挪就整个文件夹一起挪。

## 二、怎么启动

**推荐方式：双击 `启动-数据备份工具.bat`。**

为什么用这个 bat 而不是直接双击 exe？因为这个软件有两种运行模式：

| 模式 | 说明 | 怎么进入 |
| :--- | :--- | :--- |
| 真实模式 | 真正会读写你的文件、生成备份包 | 双击 `启动-数据备份工具.bat` |
| 演示模式 | 只走界面和进度条，用假数据，不动真实文件 | 直接双击 `BackupTool.exe` |

`启动-数据备份工具.bat` 会自动切到**真实模式**，所以想真备份就用它。

第一次启动如果杀毒软件或 Windows Defender 弹窗提示，选“仍要运行 / 允许”即可——这是因为程序没有数字签名，属正常现象，不是病毒。

## 三、界面怎么用

主窗口顶部有三个标签页：**① 备份、② 筛选、③ 还原**。底部是进度条和彩色日志。

### 1. 做一次备份

1. 点 **① 备份** 页。
2. “源目录”：点“浏览”，选你要备份的文件夹。
3. “输出备份包”：点“浏览”，选一个保存位置，文件名随意，比如 `我的备份.pbackup`。
4. 想省空间就勾 **哈夫曼压缩**（默认已勾）。
5. 想加密就勾 **AES-256 加密** 并输入密码（**这个密码务必记住，忘了无法还原**）。
6. 点 **开始备份**。
7. 底部进度条会走 `Scanning → Filtering → Writing → Done`，完成后日志显示“备份完成”，你选的位置就会出现那个 `.pbackup` 文件。

### 2. 做一次还原

1. 点 **③ 还原** 页。
2. “备份包”：选之前生成的 `.pbackup` 文件。
3. “还原目标目录”：选一个空文件夹（建议新建一个，别选原目录以免混淆）。
4. 如果备份时加了密，这里要输入**同一个密码**。
5. 想让同名文件被覆盖就勾 **覆盖已存在文件**。
6. 点 **开始还原**，完成后去目标文件夹检查文件是否都回来了。

### 3. 用筛选（可选）

如果只想备份部分文件，先去 **② 筛选** 页填条件，再回 ① 备份页开始。六类条件（留空=不限制）：

| 想干什么 | 填哪里 | 例子 |
| :--- | :--- | :--- |
| 只备份某类名字的文件 | 名称 | `*.docx` |
| 只备份某个子目录 | 路径-包含 | `docs` |
| 排除某个目录 | 路径-排除 | `tmp` |
| 只备份某个大小区间 | 尺寸 min/max | `1024` 到 `10485760` |
| 只备份某段时间改过的 | 时间 起/止 | `2026-07-01` |
| 按文件类型 | 类型 | `file,dir` |

多个条件同时填时是“**且**”的关系（都满足才备份）。“用户 SID”一般人不知道，留空即可。

## 四、录演示视频的推荐流程

老师要演示视频，按这个顺序录最清楚（大约 2 分钟）：

1. 双击 `启动-数据备份工具.bat` 启动。
2. 准备一个小样例文件夹，比如里面放 `a.txt`、`docs\report.txt`、一个空文件夹。
3. **演示备份**：选它 → 勾压缩 → 备份到 `demo.pbackup`，展示进度条和“备份完成”。
4. **演示还原**：新建空文件夹 `demo_restore` → 还原 → 打开对比，文件都在、内容一致。
5. **演示加密**：再备份一次并勾加密设个密码；还原时**故意输错密码**，展示程序报错拦截（这是加分点，体现健壮性）。

## 五、常见问题

**Q：双击 bat 一闪而过 / 报错缺 dll？**
A：多半是文件夹不完整或被拆散了。重新完整解压整个 zip，保持所有文件在同一个文件夹里。

**Q：杀毒软件报毒？**
A：误报（无签名的自制程序常见）。选择允许/信任即可，或临时关闭实时防护再运行。

**Q：备份很大的目录界面会不会卡死？**
A：不会。备份在后台线程跑，界面照常能动，进度条会一直更新。

**Q：`.pbackup` 文件能用压缩软件（WinRAR 等）打开吗？**
A：不能。它是本软件自定义的格式，只能用本软件的“还原”功能打开。

**Q：加密密码忘了怎么办？**
A：无法找回，也无法还原。这是加密的正常特性，请务必记住密码。
