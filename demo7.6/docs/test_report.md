# 软件测试报告

## 1. 测试环境

- 操作系统：Windows
- 编译器：MinGW g++ 8.1.0
- 构建工具：mingw32-make
- 测试入口：`tests/run_tests.ps1`

## 2. 测试范围

- 构建测试
- GUI 构建产物检查
- 基础备份和还原
- 空目录恢复
- 文件内容一致性
- 扩展名排除筛选
- 压缩备份
- 密码加密和正确密码解密
- 错误密码失败
- 归档列表查看

## 3. 测试用例

| 编号 | 用例 | 步骤 | 预期结果 |
| --- | --- | --- | --- |
| TC-01 | 构建 | 执行 `mingw32-make` | 生成 `bin/sdb.exe` 和 `bin/sdb_gui.exe` |
| TC-02 | 加密压缩备份 | 对测试源目录执行 `backup --compress --password secret` | 生成 `.sdb` 归档 |
| TC-03 | 归档查看 | 执行 `list` | 输出包含测试文件路径 |
| TC-04 | 正确密码还原 | 执行 `restore --password secret` | 文件和目录恢复成功 |
| TC-05 | 内容一致性 | 比较原始文件和还原文件内容 | 内容一致 |
| TC-06 | 筛选规则 | 使用 `--exclude-ext .tmp` | `.tmp` 文件不被还原 |
| TC-07 | 错误密码 | 使用错误密码还原 | 程序返回失败 |
| TC-08 | GUI 产物 | 检查 `bin/sdb_gui.exe` | 图形界面程序存在 |

## 4. 测试结果

自动化脚本覆盖以上用例。已在当前环境执行命令：

```powershell
mingw32-make test
```

实际输出末尾为：


## 5. 风险与改进

- 当前测试使用小规模样例目录，尚未覆盖超大文件和权限受限目录。
- 当前加密算法用于实验演示，不适合替代专业加密库。
- 后续可增加 Linux 环境测试和更多异常归档测试。
